# Hall-of-Armor
Repository for research and testing project

## Introduction
Ce dépôt est un espace libre. Pour ajouter du contenu: 
- Ajoute un dossier à la racine
- Inscris le  nom du dossier dans la section Architecture du projet, avec une phrase courte pour décrire le dossier
- Le premier commit se fait avec le message "Add {nom du dossier}"

## Architecture du projet

```text
Hall-of-Armor/
│
├── Test_Qwen/ # Test des modèles Qwen
│
├── Python_coder_0.0/ # Premier test d'équipe d'agents avec interface graphique
│

```
